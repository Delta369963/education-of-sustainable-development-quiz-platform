# 🌱 Education for Sustainable Development Quiz Platform

An interactive, responsive quiz platform designed to promote awareness and learning in **Education for Sustainable Development (ESD)** through engaging gameplay.

---

## 🎯 Project Overview

This project is a lightweight, browser-based quiz/game built using **pure HTML, CSS, and JavaScript**. It focuses on delivering an engaging user experience while keeping the implementation simple and accessible.

The application runs entirely in the browser — no backend or installation required.

---

## ✨ Features

* 🎮 Interactive quiz/game interface
* 📱 Fully responsive (mobile + desktop friendly)
* 🌙 Dark mode support
* ⏱️ Timer and performance tracking
* 🧠 Dynamic gameplay logic
* ⚡ Fast and lightweight (single-file implementation)
* 💾 Local storage support for saving progress/best scores

---

## 🧩 How It Works

* The game dynamically generates content using JavaScript
* User interactions (clicks/taps) drive the gameplay
* Scores and performance metrics are tracked in real time
* Data like best scores are stored in the browser using `localStorage`
